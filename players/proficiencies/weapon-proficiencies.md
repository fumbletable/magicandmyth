# Weapon Proficiencies

Every character starts with a number of weapon proficiency slots determined by their class. You gain additional slots as you level up.

## Slots by Class

| Class | Starting Slots | New Slot Every | Non-Proficiency Penalty |
|-------|---------------|----------------|------------------------|
| Barbarian | 4 | 3rd level | -2 |
| Fighter | 4 | 3rd level | -2 |
| Paladin | 4 | 3rd level | -2 |
| Ranger | 4 | 3rd level | -2 |
| Bard | 2 | 4th level | -3 |
| Cleric | 2 | 4th level | -3 |
| Druid | 2 | 4th level | -3 |
| Monk | 2 | 4th level | -3 |
| Thief | 2 | 4th level | -3 |
| Wizard | 1 | 6th level | -5 |

## How It Works

- Spend one slot to become proficient with a single weapon.
- Using a weapon you're not proficient in applies the **non-proficiency penalty** to attack rolls.
- Your class determines which **weapon groups** you can choose from.
- Some ancestries grant free weapon proficiencies (e.g., Elves get a free blade or bow).

## Weapon Specialization (Fighter Only)

Fighters can spend additional proficiency slots to specialize in a weapon, gaining bonus attack and damage rolls. Only fighters have access to weapon specialization.

## Weapon Groups by Class

| Class | Available Weapon Groups |
|-------|------------------------|
| Barbarian | All |
| Fighter | All |
| Paladin | All |
| Ranger | All |
| Bard | All (size limit) |
| Cleric | Clubs/Maces, Flails, Picks/Hammers, Simple Melee |
| Druid | Specific weapons only (club, dagger, dart, scimitar, sickle, sling, spear, staff) |
| Monk | Martial Arts |
| Thief | Crossbows, Finesse, Simple Melee, Simple Ranged, Whips |
| Wizard | Specific weapons only (dagger, dart, knife, sling, staff) |
