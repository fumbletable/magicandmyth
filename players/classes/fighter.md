# Fighter

*Masters of armed combat, trained in every weapon and fighting style.*

## At a Glance

| | |
|---|---|
| **Hit Die** | d10 |
| **Allowed Ancestries** | All |
| **Alignment** | Any |
| **Armor** | Any. Shields allowed. |
| **Weapon Groups** | All |

## Attribute Requirements

| Attribute | Minimum | Prime Example |
|-----------|---------|---------------|
| STR | 10 | 16 |

Failing to meet minimum: -10% XP. Meeting prime example (Human): +10% XP.

## Special Features

- **Weapon Specialization** — Fighters are the only class that can specialize in a weapon, gaining bonus attack and damage.
